# Modern Portfolyo Sitesi İnşa Promptu

Benim için GitHub Pages'de yayınlanacak modern, animasyonlu bir portfolyo sitesi oluştur.

## Teknik Gereksinimler
- Tek sayfa (single-page) HTML yapısı
- TailwindCSS CDN kullan (animasyonlar için)
- Responsive tasarım (mobil öncelikli)
- Smooth scroll davranışı
- Modern, minimalist tasarım dili

## Site Yapısı
1. **Hero Section**: Büyük başlık, kısa tanıtım, CTA butonları
2. **Hakkımda**: Fotoğraf alanı + açıklama metni
3. **Yetenekler/Teknolojiler**: İkon veya kartlarla beceriler
4. **Projeler**: Grid layout ile proje kartları (resim, başlık, açıklama, linkler)
5. **İletişim**: Sosyal medya linkleri ve iletişim formu/bilgileri
6. **Footer**: Minimal alt bilgi

## Tasarım İstekleri
- Dark mode opsiyonlu (toggle button)
- Gradient arka planlar ve modern renkler
- Hover animasyonları (scale, glow efektleri)
- Fade-in, slide-up animasyonları (intersection observer kullan)
- Glassmorphism veya modern card tasarımları
- Smooth transitions (0.3s ease)
- Sticky navigation bar

## Önemli Notlar
- Tüm metin içerikleri placeholder olarak bırak (Lorem ipsum değil, "Adınız", "Projeniz" gibi açıklayıcı placeholder'lar)
- Görsel alanlarını boş bırak veya placeholder göster
- GitHub Pages için `index.html` olarak kaydet
- JavaScript minimal olsun (sadece dark mode toggle, smooth scroll, animasyonlar için)
- TailwindCSS'in tüm utility classlarını kullan
- SEO-friendly meta taglar ekle

## Ekstra Özellikler
- Typing animation başlıkta
- Scroll progress indicator
- Back to top butonu
- Loading animasyonu (opsiyonel)
- Proje kartlarında hover'da detay gösterimi

Kodu temiz, yorumlu ve anlaşılır yaz. Tüm section'ları net şekilde ayır.
